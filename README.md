# Gabriel Calc - Calculadora Científica

Una calculadora científica moderna y profesional construida con HTML, CSS y JavaScript.

## Características

✨ **Funciones Científicas**
- Funciones trigonométricas (sin, cos)
- Constante Pi (π)
- Potencias (x^y)
- Raíz cuadrada
- Logaritmos

📱 **Compatible con Dispositivos Móviles**
- Diseño responsive
- Optimizada para pantalla táctil
- Splash screen de carga
- Vibración háptica en botones (en dispositivos compatibles)

💾 **Historial Persistente**
- Guarda automáticamente los últimos 10 cálculos
- Historial se mantiene incluso al cerrar la app
- Haz clic en un resultado anterior para recuperarlo

🎨 **Diseño Moderno**
- Tema azul profesional
- Interfaz limpia e intuitiva
- Botones con efecto visual al presionar

## Instalación como Aplicación

### En Android (Chrome)
1. Abre el link en Chrome
2. Toca los tres puntos (⋮) en la esquina superior derecha
3. Selecciona "Instalar aplicación" o "Agregar a pantalla de inicio"
4. Confirma

### En iPhone (Safari)
1. Abre el link en Safari
2. Toca el botón de compartir (↗️)
3. Selecciona "Agregar a inicio"
4. Confirma con "Agregar"

## Uso

- **Números y Operaciones**: Toca los botones para ingresar datos
- **Funciones**: Usa los botones de sin, cos, π, x^y
- **Calcular**: Presiona "=" para obtener el resultado
- **Borrar**: Presiona "BORRAR" para limpiar la pantalla
- **Historial**: Haz clic en cualquier cálculo anterior para recuperar el resultado

## Tecnología

- **HTML5**: Estructura y metadatos PWA
- **CSS3**: Diseño responsive y animaciones
- **JavaScript**: Lógica de cálculo y gestión de datos
- **LocalStorage**: Persistencia de historial
